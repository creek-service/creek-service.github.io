The v0.4.4 release of Creek is now publicly available on Maven Central and the Gradle plugin portal.

This release brings several significant improvements across the Creek ecosystem.

### Breaking Changes

 - **Java 17 required**: Creek now requires Java 17 as a minimum. Java 11 is no longer supported.
   - (Kafka): 🛠 [Upgrade Java 11 to 17 <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-kafka/pull/855){:target="_blank"}
   - (Service): 🛠 [Upgrade Java 11 to 17 <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-service/pull/455){:target="_blank"}
   - (System Test): 🛠 [Upgrade Java 11 to 17 <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-system-test/pull/681){:target="_blank"}

### New Features

 - (Kafka): 🎉 [Basic JSON Schema Serde <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-kafka/pull/459){:target="_blank"} — Creek now supports JSON Schema-based serialization/deserialization for Kafka topics.
 - (Kafka): 🎉 [Support loading schemas from another JPMS module <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-kafka/pull/485){:target="_blank"}
 - (Kafka): 🎉 [Provide way for users to register subtypes <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-kafka/pull/474){:target="_blank"}
 - (Kafka): 🎉 [Update Confluent Docker images to v3.x with KRaft <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-kafka/pull/863){:target="_blank"} — moves away from ZooKeeper-based Kafka.
 - (Service): 🎉 [Support nested resources <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-service/pull/257){:target="_blank"}
 - (Service): 🎉 [Closeable context and extensions <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-service/pull/245){:target="_blank"}
 - (System Test): 🎉 [Container starting hook <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-system-test/pull/694){:target="_blank"}
 - (System Test): 🎉 [Initialize extensions per-suite and prepare resources <i class="fas fa-external-link-alt"></i>](https://github.com/creek-service/creek-system-test/pull/378){:target="_blank"}
